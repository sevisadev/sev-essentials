Basic HTML Structure
<!DOCTYPE html><html><head><title>Page Title</title></head><body></body></html>
Centered Div
<div style="display:flex;justify-content:center;align-items:center;height:100vh;">Centered Content</div>
Button with Click Event
<button onclick="alert('Clicked!')">Click Me</button>
Link That Opens in New Tab
<a href="https://example.com" target="_blank">Open Example</a>
Simple Form
<form><input type="text" placeholder="Name"><button type="submit">Send</button></form>
Responsive Image
<img src="image.jpg" style="max-width:100%;height:auto;" alt="Responsive Image" />
Dark Mode Styles
body{background:#1e1e1e;color:#ffffff;}
Input With Placeholder
<input type="text" placeholder="Type here" />
Unordered List
<ul><li>Item 1</li><li>Item 2</li><li>Item 3</li></ul>
Simple Toggle Button
<button onclick="this.innerText=this.innerText==='ON'?'OFF':'ON'">ON</button>
